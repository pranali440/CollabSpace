# CollabSpace - AI Unified Collaborative Platform

A full-stack collaborative workspace integrating multiple tools into one platform.

## Features
- 🖊️ Real-time Whiteboard
- 💻 Collaborative Code Editor
- 📝 Shared Notepad
- 💬 Team Chat
- 📋 Kanban Task Board
- 🎥 Video Calling
- 🤖 AI Code Assistant
- 💡 Ideas Board

## Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS
- **Backend:** Spring Boot, Java, JWT
- **Database:** MySQL
- **Deployment:** Vercel (frontend), Render (backend)
